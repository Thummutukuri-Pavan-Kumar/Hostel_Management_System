# 🏨 Hostel Management System

A web-based Hostel Management System developed using **Django** and **MySQL** to streamline hostel administration. The system helps administrators, wardens, and students manage hostel activities efficiently.

## 📌 Features

### Admin Module

* Admin Login
* Dashboard Management
* Student Management
* Room Allocation
* Fee Management
* Complaint Management
* Notice Management
* Warden Management

### Warden Module

* Warden Login
* View Students
* Manage Rooms
* View Complaints
* Resolve Complaints

### Student Module

* Student Registration
* Student Login
* View Room Details
* View Notices
* View Fee Status
* Submit Complaints

## ⚙️ Installation

### Clone Repository

```bash
git clone https://github.com/Thummutukuri-Pavan-Kumar/Hostel_Management_System.git
```

### Create Virtual Environment

```bash
python -m venv test
```

### Activate Environment

```bash
test\Scripts\activate
```

### Install Dependencies

```bash
pip install -r requirements.txt
```

### Create Database

```sql
CREATE DATABASE hostel_management;
```

### Run Migrations

```bash
python manage.py migrate
```

### Start Server

```bash
python manage.py runserver
```

### Open Browser

```text
http://127.0.0.1:8000/
```
